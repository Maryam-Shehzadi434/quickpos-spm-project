## Project Overview

QuickPOS is a modern, responsive landing page for a Point of Sale system. This project demonstrates professional software development practices including:

- Git Flow branching strategy
- Jira project management with sprints
- Automated testing with PHPUnit
- CI/CD pipeline with GitHub Actions
- Code quality checks with PHPStan

## Tech Stack

- PHP 8.2+ (Form validation)
- HTML5/CSS3 (Responsive design)
- PHPUnit 10.5+ (Testing)
- GitHub Actions (CI/CD)

## Local Setup

### Prerequisites
- XAMPP (or any PHP server)
- PHP 8.2 or higher
- Composer
